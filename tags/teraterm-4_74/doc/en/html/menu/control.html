[Control] menu

Reset terminal
Resets various terminal modes.
Are you there
Sends a AYT (Are You There) telnet signal. Usually, the host sends back some message responding to it. This command is available only with a telnet connection.
Send break
Sends a break signal to the host. Refer to sendbreak macro command.
Reset port
Resets the serial port. This command is available only with a serial connection.
Broadcast command
Display the dialog for the transmission of broadcast command.
Open TEK
Opens the TEK window.
Close TEK
Closes the TEK window.
Macro
Runs a macro file.
Show Macro Window
Shows the running macro window.
