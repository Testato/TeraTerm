filewriteln

ファイルにデータを改行付きで書き込む。 

filewriteln <file handle> <data>

解説

ファイルハンドル <file handle> で指定されるファイルへ、<data> と改行文字 (CR+LF) を書き込む。

<data> が文字列型の場合、文字列を書き込む。
<data> が整数型の場合は、その値の下位バイト(0-255)を ASCII コードとみなし、その文字を書き込む。

例

filewriteln fhandle '---------cut here---------'

; NUL文字(0x00)と改行文字(CR+LF)を書き込む
filewrite fhandle 0
