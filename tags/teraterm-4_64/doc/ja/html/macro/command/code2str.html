code2str

ASCII コード値を文字列に変換する。 

code2str <strvar> <ASCII code>

解説

整数値 <ASCII code> が 1〜255 の場合、ASCII コード <ASCII code> に対応する文字を文字列変数 <strvar> に格納する。
このコマンドは <ASCII code> で表される ASCII コード列を文字列に変換し、それを <strvar> に格納する。
<ASCII code> の0でない最上位バイトがASCII コード列の1バイト目とみなされる。
<ASCII code> が0の場合、 <strvar> には長さ0の文字列 ("") が格納される。文字列の最大長は 4。

例

; 文字 "A" が変数"str" に格納される。
; "A" の　ASCII コードは$41。
code2str str $41

; 文字列 "AB" が変数 "str" に格納される。
; $41 は "A" の、$42は "B" のASCII コード。
code2str str $4142
