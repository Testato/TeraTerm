call

Calls a subroutine. 

call <label>

Remarks

Calls a subroutine beginning with the <label> line.

Example

messagebox "I'm in main." "test"
; Jump to ":sub".
call sub
messagebox "Now I'm in main" "test"
end

; Start of the subroutine.
:sub
messagebox "Now I'm in sub" "test"
; Go back to the main routine.
return

Reference

return
