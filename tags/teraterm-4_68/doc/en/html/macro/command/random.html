random

Generates random number. 

random <integer variable> <max value>

Remarks

Generates random integer within the range 0..<max number>-1 and assigns it to <integer variable>. The <max number> has to be greater than 0.

Example

; Variable val will contain integer within the range 0..99.
random val 100

; random selection
while 1
	random val 10
	int2str valstr val
	statusbox valstr 'val'
	if val=0 then
		messagebox 'The val is zero.' 'test'
	elseif val<5 then
		messagebox 'The val is less than five.' 'test'
	else
		messagebox 'The val is greater than five.' 'test'
	endif
endwhile
