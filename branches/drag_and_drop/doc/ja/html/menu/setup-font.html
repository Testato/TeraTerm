Font ([Setup] メニュー)

フォントを変更します。

フォント ダイアログボックス

フォント
変更したいフォントをリストの中から選択してください。 日本語を表示するためには日本語フォントを選択する必要があります。
スタイル
選択しても無視されます。
サイズ
フォントのサイズをポイント数で指定してください。
文字セット
表示したい文字セットをリストの中から選択してください。

注

  • Tera Term 付属のフォント Tera Special はこのダイアログボックスでは選択できません。 Tera Term によって自動的に使用されます。
  • デフォルト設定では OS で非表示に設定されたフォントは一覧に現れません。 非表示のフォントを選択するには Additiona settings ダイアログの General タブにある List hidden fonts in font dialog の設定を有効にしてください。
  • Additional settings ダイアログ の Visual タブにある Font quality でフォントの品質を選択することができます。
