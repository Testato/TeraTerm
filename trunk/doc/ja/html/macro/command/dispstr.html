dispstr

データを表示する。 

send <data1> <data2>....

解説

<data> が文字列型の場合、文字列をウインドウに表示する。
<data> が整数型の場合は、その値の下位バイト(0-255)を ASCII コードとみなし、その文字を表示する。

例

dispstr 'ABC'

; "ABC" を表示する。("A" の ASCII コードは65)
dispstr  65 66 67
myname='Tera Term'
dispstr 'My name is ' myname '.'

; ウィンドウの表示内容をクリアする
dispstr #$1B"[2J"

; ウィンドウの10行目20桁目に Message と表示する
dispstr #$1B"[10;20HMessage"

; ウィンドウの位置を 100,200 に移動する
dispstr #$1B"[3;100;200t"
