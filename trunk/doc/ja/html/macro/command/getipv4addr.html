getipv4addr

自端末のIPv4アドレスを取得する。 

getipv4addr <string array[0]>

パラメータ

配列 <string array[0]>
IPアドレスを格納する文字型配列。最大10個まで。

返り値

システム変数 <result>
IPアドレスの個数が格納される。
IPアドレスが存在しない場合、0が格納される。

例

strdim ipaddr 10

getipv4addr ipaddr[0]
N = result - 1
for i 0 N
	messagebox ipaddr[i] "IPv4 address"
next
