Font ([Setup] メニュー; ロシア語モード)

フォントを変更します。

"Font setup" ダイアログボックス

Font
変更したいフォントをリストの中から選択してください。 日本語を表示するためには日本語フォントを選択する必要があります。
Size
フォントのサイズをポイント数で指定してください。
Enable bold style
Bold (強調)属性の文字を bold スタイルのフォントで表示するようにします。 この場合、画面にゴミが残ったり、 細かい文字がつぶれて読みにくくなったりすることがあります。
Font char set
指定されたフォントの文字コードを、Windows (CP 1251), KOI8-R, CP 866, ISO 8859-5 の中から選択します。

注:
Tera Term 付属のフォント Tera Special はこのダイアログボックスでは選択できません。 Tera Term によって自動的に使用されます。
