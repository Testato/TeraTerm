show

Changes the displaying of MACRO dialog box. 

show <show flag>

Remarks

Minimizes MACRO, if <show flag> is zero.
Restores MACRO, if <show flag> is greater than zero.
Hides MACRO, if <show flag> is less than zero.

Example

; Minimize MACRO.
show 0

; Restore MACRO.
show 1

; Hide MACRO.
show -1
