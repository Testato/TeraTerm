strspecial

Converts special characters.

Format

strspecial <strvar> [<strval>]

Parameters

string valiable <strvar>
string <strval> (optional)

Return Value

System valiable <result>

Remarks

Converts the following characters in the <strvar> to special characters.

input output
\\ "\"
\n line feed
\t tab

Example

str = "AB\tCD\nEF\\nGH"
strspecial str
messagebox str "special"

strspecial str "AB\tCD\nEF\\nGH"
messagebox str "special"
