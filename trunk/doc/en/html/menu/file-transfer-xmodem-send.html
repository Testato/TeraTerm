XMODEM Send ([File] Transfer menu)

Sends a file to the host with the XMODEM protocol. Activate the receive command of the host's XMODEM before executing this command.

"XMODEM Send" dialog box

File Name

Enter or select the file name of the existing file to be sent.

Option

1K
Selects this option when packet size(1K) is sent.
Type of error detection(Checksum/CRC) can be only specified at receiver side.
