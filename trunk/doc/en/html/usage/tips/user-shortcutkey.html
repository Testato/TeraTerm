User shortcut keys

You can define shortcut keys for the following functions by editing items in the [Shortcut keys] and [VT function keys] sections of the keyboard setup file see also KEYCODE.TXT): 

[Shortcut keys]
Item in the         Function                                Default key


EditCopy            [Edit] Copy command *1)                 Ctrl+Insert
EditPaste           [Edit] Paste command *1)                Shift+Insert
EditPasteCR         [Edit] Paste<CR> command *1)            none
EditCLS             [Edit] Clear screen command             none
EditCLB             [Edit] Clear buffer command             none
ControlOpenTEK      [Control] Open TEK command              none
ControlCloseTEK     [Control] Close TEK command             none
LineUp              Scrolls up screen by 1 line             Ctrl+Up-arrow
LineDown            Scrolls down screen by 1 line           Ctrl+Down-arrow
PageUp              Scrolls up screen by 1 page             Ctrl+PageUp
PageDown            Scrolls down screen by 1 page           Ctrl+PageDown
BuffTop             Scrolls screen to buffer top            Ctrl+Home
BuffBottom          Scrolls screen to buffer bottom         Ctrl+End
NextWin             Moves to the next Tera Term window      Ctrl+Tab
PrevWin             Moves to the previous Tera Term window  Ctrl+Shift+Tab
NextShownWin        Moves to the next Tera Term window      none
		    (except minimized window)
PrevShownWin        Moves to the previous Tera Term window  none
		    (except minimized window)
LocalEcho           Toggles the local echo status           none


[VT function keys]
Item in the         Function                                Default key


Hold                Locks/unlocks the port for receiving    none
Print               [File] Print command *1)                none
Break               Sends a break signal *1)                none

*1) System shortcut keys are assigned for these functions.

If you want to assign Ctrl+V for the [Edit] Paste command, edit the EditPaste item in the [Shortcut keys] section of the keyboard setup file: 

EditPaste=1071

(In the case of IBM PC/AT keyboard)

You can also define user shortcut keys for all menu commands by using user keys at the [User keys] section.
