TTXResizeMenu -- Tera TermのVTウィンドウのサイズを変更するメニューを表示する

機能:
  Tera Termのメニューに、Resizeメニューを追加します。
  Resizeメニュー内の項目が選択されると、そのサイズにTera TermのVTウィンドウの
  サイズが変更されます。

解説:
  TTXResizeWinをメニュー形式にして、複数のサイズの中から選べるようにした
  だけのものです。
  リサイズ処理の詳細については、TTXResizeWinの解説を参照してください。

  メニュー内の項目を変更したい場合は、TERATERM.INIに[Resize Menu]セクションを
  作成し、ResizeMenuN = X, Y の形式で指定して下さい。
  X または Y のいずれかが 0 の時は、その要素は変更されません。例えば、0,37 の
  時は、横幅は変更せずに縦幅を 37 に変更します。
  メニューの項目数は最大で 20 です。最大数を増やしたい時は TTXResizeMenu.c の
  MAX_MENU_ITEMS を増やしてからビルドし直してください。

  TERATERM.INIに設定が無い場合、デフォルトの設定が使われます。
  デフォルトの設定は作者がよく使うサイズで、以下の理由で決めました。

  1. 80x37  -- 横幅はVTの標準、縦幅はXGA(1024x768)で画面いっぱい。
               ノートPCのモニタ用。
  2. 120x52 -- 横幅は広め、縦幅はSXGA(1280x1024)で画面いっぱい。
               外部モニタやrdesktop接続時用。
  3. 80x24  -- VTの標準サイズ。
  4. 110x37 -- 80x37の幅が広い版。

  いずれも、フォントがTerminal 14で、WindowsXPのLunaテーマ、タスクバーを
  自動的に隠さない設定の時の話です。

設定例:
  [Resize Menu]
  ResizeMenu1 = 80, 37
  ResizeMenu2 = 120, 52
  ResizeMenu3 = 80, 24
  ResizeMenu4 = 110, 37
