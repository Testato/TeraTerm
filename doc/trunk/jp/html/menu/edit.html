[Edit] メニュー 
Copy

マウスで選択した文字列を、クリップボードへコピーします。
注意: 通常は、このコマンドを使う必要はありません。マウスで文字を選択すると、自動的にクリップボードへコピーされます。選択されている文字列を再コピーする場合にこのコマンドを使用してください。

Copy table

マウスで選択した文字列をクリップボードへコピーします。コピーされた文字列が複数の単語で構成されている場合、単語間の各空白文字列はそれぞれ一つの Tab 文字に変換されます。このコマンドにより画面上の表をコピーして、それを Excel などの表計算ソフトに貼り付けることができます。

Paste

クリップボードの中の文字列をホストへ送出します。

Paste

クリップボードの中の文字列をホストへ送出し、続いて改行コードも送出します。

Clear screen

画面を消去します。

Clear buffer

スクロールバッファーの内容と画面を消去します。
