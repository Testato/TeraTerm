[Control] メニュー

Reset terminal
ターミナルの各種モードをリセットします。
Are you there
Telnet の AYT (Are You There) コマンドを送出します。通常ホストはこれに呼応して何等かのメッセージを送り返してきます。Telnet 接続のときだけ使用できます。
Send break
ブレーク信号を送出します。
Reset port
シリアルポートをリセットします。シリアルポート接続のときだけ使用できます。
Open TEK
TEK window を開きます。
Close TEK
TEK window を閉じます。
Macro
マクロを実行します。
