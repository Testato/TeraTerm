Window ([Setup] メニュー)

"Window setup" ダイアログボックス

Title
ウインドウののタイトル。
Cursor shape
カーソルの形。Block, Vertical line, Horizontal line の3つから選べます。
Hide title bar
タイトルバーとメニューバーを消します。ウィンドウを移動させるには Alt キーを押しながら右マウスボタンでウィンドウをドラッグしてください。
ウインドウを最小化させるには Alt キーを押しながら左マウスボタンを押してください。Ctrl キーを押しながら左マウスボタンを押すとポップアップメニューが現れます。
Hide menu bar
メニューバーを消します。Ctrl キーを押しながら左マウスボタンを押すとポップアップメニューが現れます。
Full color
フルカラーエミュレーションを行います。このモードでは、Bold(強調)/Blink(点滅)属性が文字/背景色の属性の一部として使用され、エスケープシーケンスにより指定可能な文字/背景色が8色から16色に増えます。
Scroll buffer
スクロールバッファーを使用可能にします。また、スクロールバッファの行数が設定できます。行数は 800 (Tera Term) または 10000 (Tera Term Pro) 以上には設定できません。メモリが足りない場合は適当に減らされます。
Color
Text
文字の色
Background
背景の色
Attribute
色を設定したい文字の属性 (Normal, Bold, Blink)
Reverse
文字の色と背景の色を交換します。
