XMODEM Send ([File] Transfer メニュー)

XMODEM プロトコルでファイルを受信します。その前にホスト側の XMODEM の送信コマンドを起動してください。

File Name

受信するファイルのファイル名を入力または選択してください。

Option

Checksum/CRC/1K
XMODEM プロトコルのオプション (Checksum/CRC/1K) を選択してください。ホスト側のオプションと同じにするか、よくわからない場合は Checksum を選択してみてください。
Binary
バイナリファイルを受信する場合は選択してください。逆にテキストファイルを受信する場合は選択しないでください。
