setdlgpos

形式：
    setdlgpos <x> <y> 

"inputbox", "messagebox", "passwordbox", "statusbox" コマンドで表示されるダイアログボックスの初期位置を変更する。
また、ステータスダイアログボックスが表示されている場合、setdlgpos コマンドはそのダイアログボックスを移動させる。

 と  は画面の座標 (x,y) を表わす。原点 (0,0) は画面の左上の隅の位置。



例：
     setdlgpos 0 0
     messagebox 'Message' 'Title'                    左上の隅にダイアログボックス

     setdlgpos 0 200                                  ステータスダイアログボックスを開く
     statusbox 'Message' 'Title'
     for i 0 200
       setdlgpos i 200
     next                                              ステータスダイアログボックスを移動
