getver

TeraTermのバージョンを取得する。 

getver <strvar> [<version>]

解説

TeraTermのバージョンを <strvar> に格納する。
（｛<version>｝が省略された場合、本マクロコマンドが実装され たバージョンが指定される）
また、コマンド実行結果をシステム変数resultに格納する。 変数 result の値の意味は以下のとおり。

変数 result の値の意味は以下のとおり。

値 意味
0 正常に値が <strvar> に格納された。
1 TeraTermのバージョンが<version>より古い。
2 TeraTermのバージョンが<version>と同じ。
3 TeraTermのバージョンが<version>より新しい。

例


getver version

getver version '4.56'
