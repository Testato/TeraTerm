statusbox

形式：
    statusbox <message> <title> [<special>(オプション)]

ステータスダイアログボックスが表示されていない場合、それを表示する。
ダイアログボックスのメッセージを <message> に、タイトルを <title> に変える。

"setdlgpos" コマンドはステータスダイアログボックスの位置を変える。
"closesbox" コマンドはステータスダイアログボックスを閉じる。

<special> が 0 でない場合は、<message> に含まれる以下の文字列は特殊文字として扱われます。
<special> が指定されない場合は 1 が使用されます。 

\\    "\"
\n    改行
\t    タブ

例:
    setdlgpos 200 200            表示位置を指定
    statusbox 'Message' 'Title'  ダイアログボックスを表示
    pause 3
    setdlgpos 0 0                ダイアログボックスを移動
    pause 3
    closesbox                    ダイアログボックスを閉じる
