yesnobox

形式：
    yesnobox <message> <title>

Dalog boxを開き、ユーザーに「はい」/「いいえ」を選択させる。
文字列 <message> はダイアログボックスに表示されるメッセージ、文字列 <title> はダイアログボックスのタイトル。 「はい」ボタンが押された場合、システム変数 resultに1が格納される。「いいえ」ボタンが押された場合、result に0が格納される。

<message> に含まれる以下の文字列は、特殊文字と扱われます。 

\\    "\"
\n    改行
\t    タブ

例：
     yesnobox 'もう一回やりますか?' 'Tera Term'
     if result goto retry
     end
