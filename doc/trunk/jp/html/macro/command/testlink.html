testlink

形式：
    testlink 

    
現在のリンクおよび接続の状態を報告する。
リンクと接続の状態に応じて以下のうちの一つの値がシステム変数 result に格納される。



    
値							状態

    
0							Tera Term とリンクされていない。
1							ホストへの接続はされていないが、Tera Term へのリンクはされている。
2							リンクおよび接続両方ともされている。

参照：
      "closett"
      "connect"
      "disconnect"
      "unlink"



    
例：
    testlink                                          もし MACRO が Tera Term にリンクされていなければ
    if result=0 connect 'host'                      connect コマンドを実行
