flushrecv

形式：
    flushrecv

MACRO の受信バッファーの内容を消す。


ホストから受信した文字は MACRO に転送され、MACRO の受信バッファーに保存される。受信文字を処理するコマンド("wait" コマンド等)はそれらの
文字をバッファーから読み出す。バッファーの中の文字は受信文字を処理するコマンドがそれらを読みだすか、バッファーがあふれるか、flushrecv コマンド
がバッファーの内容を消すまで保持される。

バッファーの中に古い文字がたまっているために受信文字を処理するコマンドの結果が予期しないものになることがあり、flushrecv コマンドによってそれを
防ぐことができる。
