include

形式：
    include <include file name> 

インクロードファイルに移る。

例：

-----メインファイル"main.ttl"------

   i=10
   :loop
   include 'sub.ttl'                               インクルードファイルへ移る。
   if i>=0 goto loop
   end

-----メインファイル終わり----------


-----インクルードファイル"sub.ttl"----

   if i<0 then
     messagebox 'error!' 'sub'
     exit　　　　　　　　　　　　　　　　　　　　　メインファイルへ戻る。
   endif
   i=i-1　　　　　　　　　　　　　　　　　　　　　 メインファイルへ戻る。

-----インクルードファイル終わり-------
