cygconnect

Cygwinへ接続する。 

cygconnect

解説

Cygwin（Windowsで動くLinuxライク環境）へ接続する。

Cygwinのインストール先は"cyglaunch.exe"が自動的に検索を行い、cygwin1.dllへのパスを設定した状態で、CygTermを呼び出す。

例


prompt = '$'

cygconnect
wait prompt

sendln 'pwd'
wait prompt

sendln 'uptime'
wait prompt
