TELNET プロトコルに関連付ける

実行するアプリケーションを、下記のように指定します。

"C:\Program Files\teraterm\ttermpro.exe" /T=1 /nossh %1

コマンドラインオプションに "/T=1" を指定することで、接続が TELNET であることを明示的に ttermpro.exe に通知します。

TTSSH がインストールされていて TERATERM.INI で SSH が有効だと、TeraTerm は SSH 接続を開始してしまいます。
コマンドラインオプションに "/nossh" を指定することで、TTSSH に "SSHを無効にする" ことを通知します。
