ZMODEM のヒント 
ZMODEM は、7 bit 回線では使用できません。

シリアルポートの場合

Data が 7 bitだと、ZMODEM が使えません。[Setup] Serial port で Data を 8 bit にしてください。モデム(ホスト)側の設定も同じにしてください。

Telnet の場合

もし正しく転送できない場合は制御文字をすべてエスケープしてみてください。それでもだめな場合はバイナリオプションで telnet 接続してみてください。
バイナリオプションで接続するには、

コマンドラインに /B オプションをつけるか、

Tera Term 設定ファイルを書き換えてください。

→ Telnet のバイナリオプション
