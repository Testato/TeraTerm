[Setup] メニュー

Terminal...
端末エミュレーションの設定をします。
Window...
画面表示の設定をします。
Font...
フォントを変更します
Keyboard...
キーボードの設定をします。
Serial port...
シリアルポートの設定をします。
Proxy...
プロキシの設定をします。
SSH...
SSH 全般の設定をします。
SSH Authentication...
SSH 認証の設定をします。
SSH Forwarding...
SSH ポートフォワーディングの設定をします。
SSH Keygenerator...
SSH のキーペア認証で用いるキーの生成を行います。
TCP/CP...
TCP/IP の設定をします。
General...
その他の設定をします。
Save setup...
現在の設定を Tera Term 設定ファイルに保存します。
Restore setup...
設定ファイルの内容にしたがって Tera Term を設定しなおします。
Load key map...
キーボード設定ファイルの内容にしたがってキーボードを設定しなおします。
