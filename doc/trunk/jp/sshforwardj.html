SSH Forwarding（[Setup] メニュー）

"TTSSH: Forwarding " ダイアログボックス

SSH 通信を使ってポート転送を行う設定ができます。
SSH 経由で暗号化して、既存のプロトコルをセキュアな通信を行わせる場合に指定します。

Port Forwarding

Add...
Forward する方向、ポート番号を指定して一覧に追加します。
Edit...
選択された Forwarding の設定を編集します。
Remove...
選択された Forwarding の設定を削除します。

X Forwarding

Display remote X application on local X server
このチェックボックスをオンにすると、サーバ上の X クライアントとPC上の X サーバの間の通信が暗号化されます。
