SSH Authentication（[Setup] メニュー）

"TTSSH: Authentication Setup" ダイアログボックス

User name
SSH 接続で用いるユーザ名を指定します。
Use plain passwd to log in
パスワード認証を用いる場合に指定します。SSH2 接続の場合では "keyboard-interactive (SSH2)" を指定できます。この場合、通信路にはパスワードそのものは流れません。（通常のパスワード認証の場合は、暗号化された上でパスワードそのものが流れます。）
Use RSA/DSA key to login
あらかじめ作成した RSA/DSA 鍵を用いた認証方法を選択します。
Use rhost to login (SSH1)
ホストの設定で rhost が利用できる場合に選択します。
Use challenge/response to log in (SSH1:TIS, SSH2:keyboard-interactive)
チャレンジ／レスポンス認証を用いる場合に指定します。
