ユーザー・ショートカット・キー
キーボード設定ファイルの[Shortcut keys] と [VT function keys] セクションの項目を書き換えると、以下のショートカット・キーを自分の好きなキーに割り当てることができます。詳しくは、KEYCODEJ.TXTを読んでください。


[Shortcut keys]
セクションの項目        機能                                 初期設定


EditCopy           [Edit] Copy コマンド*1)               Ctrl+Insert
EditPaste          [Edit] Paste コマンド *1)             Shift+ nsert
EditPasteCR        [Edit] Paste<CR> コマンド *1)           なし
EditCLS            [Edit] Clear screen コマンド             なし
EditCLB            [Edit] Clear buffer コマンド             なし
ControlOpenTEK    [Control] Open TEK コマンド              なし
ControlCloseTEK   [Control] Close TEK コマンド             なし
LineUp             一行スクロールアップ                    Ctrl+↑
LineDown           一行スクロールダウン                    Ctrl+↓
PageUp             一ページスクロールアップ                Ctrl+PageUp
PageDown           一ページスクロールダウン                Ctrl+PageDown
BuffTop            バッファの先頭へスクロール              Ctrl+Home
BuffBottom         バッファの最後へスクロール              Ctrl+End
NextWin            次の Tera Term ウィンドウへ移動         Ctrl+Tab
PrevWin            前の Tera Term ウィンドウへ移動         Ctrl+Shift+Tab
LocalEcho          ローカルエコーを on/off にする。　　　　 なし


[VT function keys]
セクションの項目        機能　　　　　　　　　　　　　　　　　　初期設定


Hold                   受信ポートをロック/ロック解除           なし
Print                  [File] Print コマンド *1)               なし
Break                  ブレーク信号送出 *1)                    なし

*1) これらの機能にはシステム・ショートカット・キーが割り当てられています。

例えば、Ctrl+V を [Edit] Paste に割り当てたい場合は、
キーボード設定ファイルの [Shortcut keys] セクションで、

EditPaste=1071

としてください (IBM PC/AT キーボードの場合)。

また [User keys] セクションでは、ユーザーキーを使ってすべてのメニューコマンドに対してショートカット・キーを設定することができます
