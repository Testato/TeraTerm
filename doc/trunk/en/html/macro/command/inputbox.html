inputbox

Format:
    inputbox <message> <title> [<default>(optional)]

Displays a dialog box prompting user to input a string.
The <message> is displayed in the dialog box. The <title> is displayed as the dialog box title. The <default> is default string on edit control. The string entered by the user is returned in the system variable "inputstr".

Following strings in <message> is treated as spacial character. 

\\    "\"
\n    line feed
\t    tab

Example:
    inputbox 'Password:' 'Login'
    sendln inputstr
