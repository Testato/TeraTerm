statusbox

Format:
    statusbox <message> <title>

Displays the status dialog box if it has not been displayed yet. Changes the message to <message> and title to <title>.

The "setdlgpos" command changes the position of status dialog box.
The "closesbox" command closes the status dialog box. 

Example:
  setdlgpos 200 200		Set the initial position.
  statusbox 'Message' 'Title'	Display the status dialog box.
  pause 3	
  setdlgpos 0 0			Move the dialog box.
  pause 3	
  closesbox			Close the dialog box.
