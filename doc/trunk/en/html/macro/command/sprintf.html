sprintf

Format:
    sprintf FORMAT [ARGUMENT] ... 

FORMAT controls the output as in C language printf with ARGUMENTs converted to proper type first.
The output string is stored in the string variable <inputstr>.
And also, the argument counts are interpreted only up to five.

Example:
    sprintf 'TeraTerm 4.%d' 51
    messagebox inputstr 'sprintf test(1)'

    sprintf 'Windows %d (+%s)' 2000 'SP4'
    messagebox inputstr 'sprintf test(2)'

    sprintf '%s=%d %s=0x%x' 'dec' 10 'hex' 33
    messagebox inputstr 'sprintf test macro(3)'
