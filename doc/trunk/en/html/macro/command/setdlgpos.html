setdlgpos

Format:
    setdlgpos <x> <y>

Changes the initial position for dialog boxes opend by the "inputbox", "messagebox", "passwordbox" and "statusbox" commands. If the status dialog box is displayed, the "setdlgpos" command also moves the dialog box.

and specify the position (x,y) in the screen coordinate. The origin (0,0) is upper left corner of the screen. 

Example:
  setdlgpos 0 0	
  messagebox 'Message' 'Title'		Message box at the upper left corner.

  setdlgpos 0 200			Open the status box.
  statusbox 'Message' 'Title'	
  for i 0 200	
    setdlgpos i 200			Moves the status box.
