testlink

Format:
    testlink

Reports the current link and connection status. The system variable "result" is set to one of the following values depending on the link and connection status: 

Value	Status


0	Link to Tera Term has not been made.
1	Connection to the host has not been made, but link to Tera Term has been made.
2	Both connection and link have been made.

See also:
	"closett"
	"connect"
	"disconnect"
	"unlink"

Example:
	testlink	
	if result=0 connect 'host'			If MACRO is not linked to Tera Term, 
							execute the connect command.
