execcmnd

Executes a TTL statement. 

execcmnd <statement>

Remarks

Executes a TTL statement expressed by the string <statement>.

Example

; Execute the statement "send 'abc'".
execcmnd "send 'abc'"
; Stores 1 to the variable 'a'.
execcmnd "a=1"

for x 1 10      ;creates variables arr1 thru arr10 and assigns them 1 thru 10
  tmp='arr'
  int2str ix x
  strconcat tmp ix
  strconcat tmp '='
  strconcat tmp ix
  execcmnd tmp
next
