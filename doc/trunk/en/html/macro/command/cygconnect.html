cygconnect

Connects to Cygwin. 

cygconnect

Remarks

This command enables TeraTerm to connect to Cygwin -a Linux-like environment for Windows.

The "cyglaunch.exe" application automatically searchs Cygwin installation directory, and calls CygTerm after configuring PATH environment variable added cygwin1.dll.

Example


prompt = '$'

cygconnect
wait prompt

sendln 'pwd'
wait prompt

sendln 'uptime'
wait prompt
