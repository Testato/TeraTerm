include

Format:
    include <include file name>

Moves control to the include file. 

Example:
----- Main file "main.ttl" ------
	i=10	
	:loop	
	include 'sub.ttl'		Move to the include file.
	if i>=0 goto loop	
	end	
----- End of "main.ttl" ---------

----- Include file "sub.ttl" ----
	if i<0 then	
	  messagebox 'error!' 'sub'	
	  exit				Go back to the main file.
	endif	
	i = i - 1			Go back to the main file.
----- End of "sub.ttl" ----------
