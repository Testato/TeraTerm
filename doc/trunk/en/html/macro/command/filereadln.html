filereadln

Format:
    filereadln <file handle> <strvar>

Reads a line from the file specified by <file handle>. The line is written into the string variable <strvar>. The file pointer is moved to the beginning of the next line. If the file pointer reaches the end of the file while reading the line, the system variable "result" is set to 1. Otherwise, "result" is set to zero. 

Example:
	fileopen fhandle 'test.txt' 0		Open a file.
	:loop	
	filereadln fhandle line			Read a line from the file.
	if result goto fclose	
	messagebox line 'test.txt'		Display the line.
	goto loop				Repeat until the end of the file.
	:fclose	
	fileclose fhandle			Close the file.
