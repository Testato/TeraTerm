Expressions and operators

Expressions consist of constants, variables, operators, and parentheses. Constants and variables must be of the integer type. The value of an expression is also an integer. The value of a relational expression (formed using relational operators) is 0, if it is true, or 1 if false.

The following are operators: 

Category             Precedence    Operators


bitwise negation     1, high       not ~
logical negation     1             !
multiplicative       2             * / %
bitwise conjunction  2             and &
additive             3             + -
bitwise disjunction  3             or xor | ^
relational           4             = <> < > <= >=
logical conjunction  5             &&
logical disjunction  6, low        ||

Note: the value of expression A % B is the remainder of A / B.
      "and", "or", "xor" and "not" is bitwise operator, not logical operator.
      ~, !, &, |, ^, && and || is available in version 4.53 or later.

Example:
    1 + 1
    4 - 2 * 3      The value is -2.
    15 % 10        The value is 5.
    3 * (A + 2)    A is an integer variable.
    A and not B
    A <= B         A and B are integer variables. The value is 1,
                   if the expression is true, or 0 if false.
