Expressions and operators

Expressions consist of constants, variables, operators, and parentheses. Constants and variables must be of the integer type. The value of an expression is also an integer. The value of a relational expression (formed using relational operators) is 1, if it is true, or 0 if false.

The following are operators:

Precedence Operators Category
1, high not ~ bitwise negation
! logical negation
+ plus unary
- minus unary
2 * / % multiplicative
3 + additive
- additive
4 >> << >>> shift
5 and & bitwise disjunction
6 xor ^ bitwise disjunction
7 or | bitwise disjunction
8 < > <= >= relational
9 = == <> != relational
10 && logical conjunction
11, low || logical disjunction

the value of expression A % B is the remainder of A / B.
"and", "or", "xor" and "not" is bitwise operator, not logical operator.
~, !, &, |, ^, && and || is available in version 4.53 or later. 

Example:
    1 + 1
    4 - 2 * 3      The value is -2.
    15 % 10        The value is 5.
    3 * (A + 2)    A is an integer variable.
    A and not B
    A <= B         A and B are integer variables. The value is 1,
                   if the expression is true, or 0 if false.
