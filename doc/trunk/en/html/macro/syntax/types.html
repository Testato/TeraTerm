Types

TTL have two kinds of data types:

Integer
    Signed 16 bit, from -32767 to 32768, in case of TTMACRO.EXE for Windows 3.1. Signed 32 bit, from -2147483648 to 2147483647, in the case of TTPMACRO.EXE for Windows 95/98/NT4.0/2000/XP.

Character string
    A sequence containing any character except NUL. The maximum length of a string is 255.
