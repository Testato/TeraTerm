Command line


TTPMACRO.EXE [/I] [/V] [<macro file> [<second param>] [<third param>]]

where:
	/I		Start MACRO in iconized state.
	/V		Start MACRO in hidden (invisible) state.
	<macro file>	Macro filename.
	<second param>	Character string stored in the system variable "param2".
	<third param>	Character string stored in the system variable "param3".

See "Variables" for the system variables "param2" and "param3".
