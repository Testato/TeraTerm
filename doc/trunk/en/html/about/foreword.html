Foreword

Dear TeraTerm user,

This topic briefly covers "UTF-8 TeraTerm Pro with TTSSH2" creation and development history.

First of all I would like to thank many people involved in TeraTerm development starting Takashi Teranishi who created the first version of TeraTerm back in 1994. TeraTerm is pronounced `tera ta-mu' and also, the abbreviation is `TT'. TeraTerm has been developed by original author from 1994 to 1998. TeraTerm is a terminal emulator for Microsoft Windows, supports serial port and telnet connections. It also has a built in Macro scripting language. It was often used to do automatic operation of personal computer communication at that time.

TeraTerm was released as freeware and won Japan software competition called by "Online software grand prize 97" of Madonomori in 1997. TeraTerm also supports the plugin mechanism and has many other useful plugins. One of all, TTSSH, which name stands for An SSH Extension to Teraterm, is famous for everyone. TTSSH was developed by Robert O'Callahan back in 1998. If TTSSH plugin is built in TeraTerm, SSH connection with SSH1 protocol was able to done. Unfortunately, Robert's homepage is not found now.

TeraTerm latest version is 2.3 and supporting OS is Windows 95 and NT. Unusually in the first time of Windows series, the source code of both TeraTerm and TTSSH are opened. But TeraTerm is not open source software because a user must get the permission distributing binary files from author, that is, Takashi Teranishi.
In this document, TeraTerm by Takashi Teranishi is called by Original TeraTerm or TeraTerm 2.3.

TeraTermは長らく日本におけるターミナルエミュレータの代名詞として、ユーザに親しまれてきましたが、1998年に開発が終了しているために、最新のWindowsへの対応や新しいプロトコルへのサポートがなされていませんでした。ソースコードは公開されているため、ユーザの手によりIPv6版や半透明版などの派生バージョンが登場しました。しかし、原作者へ連絡が付かない状態であったために、これらの派生バージョンはパッチ形式で配布されていました。なにより、完全なバイナリを配布するには原作者である寺西高氏の許可を取る必要があるために、TeraTermの開発を第三者が引き継ぐことが難しい状況となっていました。

2004年9月に寺西高氏に連絡が取れ、同年11月にSourceForge.jpへプロジェクトを立ち上げ、TeraTermのオープンソース化を実現しました。TeraTermがオープンソース・ソフトウェアであるとするために、BSDライセンスの下に"TeraTerm Project"が発足しました。このプロジェクトはTeraTerm後継版として開発が進められており、「UTF-8 TeraTerm Pro with TTSSH2」と呼ばれています。
UTF-8 TeraTerm Pro with TTSSH2のコミュニティは日本だけではなく、海外（カナダ）にも存在し、日本発の世界へ向けたオープンソース・ソフトウェアとして活動が続けられています。
