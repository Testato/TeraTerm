listbox

Displays a list box, and ask a item to user. 

listbox <message> <title> <item1> [<item2>] [...] [<item10>]

Parameters

string <message>
It is displayed in the list box.
string <title>
It is displayed as the list box title.
string <itemN> (N=1-10)
It is displayed as the list box items.

Return Value

System variable <result>
If the user selects one of items, it is set to from 0 to N-1.
If the user cancels, it is set to -1.

Example

listbox 'Select your favorite food.' 'Confirmation' 'Banana' 'Apple' 'Orange'
sprintf2 var "%d" result
messagebox var "result"
; result
;   -1: Cancel
;    0: Banana
;    1: Apple
;    2: Orange
