getipv4addr

Retrieves local IPv4 address(es) 

getipv4addr <string array>

Parameters

Array <string array>
IP addresses are stored in the string array.

Return Value

System variable <result>
It is set to the IP address number.
If the IP address is nothing, it is set to 0.
When the IP address can not retrieve, it is set to -1.

Example

strdim ipaddr 10

getipv4addr ipaddr
if result = -1 then
	messagebox 'Error' 'IPv4 address'
elseif result = 0 then
	messagebox 'Not found' 'IPv4 address'
else
	N = result - 1
	for i 0 N
		messagebox ipaddr[i] "IPv4 address"
	next
endif
