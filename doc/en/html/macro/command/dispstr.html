dispstr

Displays data. 

dispstr <data1> <data2>....

Remarks

Causes Tera Term to display characters on the window.
If <data> is a string, the string is displayed on the window.
If <data> is an integer, its lowest-order byte (0-255) is regarded as an ASCII code of the character, and the character is displayed on the window.

Example

dispstr 'ABC'

; Display "ABC". (ASCII code of the character "A" is 65.)
dispstr  65 66 67
myname='Tera Term'
dispstr 'My name is ' myname '.'

; 
dispstr #$1B"[2J"

; 
dispstr #$1B"[10;20HMessage"

; 
dispstr #$1B"[3;100;200t"
