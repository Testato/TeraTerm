bringupbox

Brings the status box to the top of the Z order. 

bringupbox

Parameters

None.

Return Value

None.

Example

step = 0
while 1
  call show_step
  sendln 'echo do something'
  wait '$'
  pause 2

  bringupbox

endwhile
end

:show_step
step = step + 1
gettime timestr "%Y/%m/%d-%H:%M:%S"
sprintf2 s "counter %d" step
statusbox timestr s
return
