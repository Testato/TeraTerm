expandenv

Expands environment variables.

Format

expandenv <strvar> [<strval>]

Parameters

string valiable <strvar>
string <strval> (optional)

Return Value

System valiable <result>

Remarks

Example

str = "%WINDIR%\notepad.exe"
expandenv str
messagebox str "env"

expandenv str "%WINDIR%\notepad.exe"
messagebox str "env"
