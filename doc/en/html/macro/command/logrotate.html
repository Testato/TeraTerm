logrotate

Rotates a log file. 

logrotate 'size' <num>
logrotate 'halt'

Parameters

'size' <num>
When a log file size is over <num> byte, the log file is be rotated.
The <num> value must be larger than 128.
'halt'
The log rotation is stopped.

Example

getspecialfolder str "Desktop"
changedir str
logopen 'teraterm_sample.log' 0 0
; Start log rotation
logrotate 'size' 2048
pause 600
; Stop log rotation
logrotate 'halt'
pause 600
logclose
