Proxy([Setup] menu)

"Proxy Setup" dialog box

Type
Configure Proxy type
  • Winthout Proxy: Do not use Proxy
  • HTTP: Use HTTP Proxy
  • TELNET: Use TELNET Proxy
  • SOCKS4: Use SOCKS4 Proxy
  • SOCKS5: Use SOCKS5 Proxy
Hostname
Specify Proxy Server
Port number
Specify port number for Proxy server
Username
When an authentication is required for Proxy server, specify user name
Passwd
When an authentication is required for Proxy server, specify password
Other setup
Move to "Other Setup" menu

"Other Setup" menu

Common setup

Connection Timeout
The connection to Proxy will be reset if a communication with Proxy (including keep-alive) is lost for this time (seconds)
Proxy connection log
The communication with Proxy server will be logged in this file

SOCKS proxy setup

Specify a name resolution method by Name

auto resolve
Automated resolution
resolve remote
Resolution is done by SOCKS server
resolve local
Resolution is done by PC where Tera Term is running

TELNET Proxy setup

Configure TELNET Proxy parameters

Hostname prompt
Specify prompt characters for Host name
Username prompt
Specify prompt characters for User name
Passwd
Specify prompt characters for Password
Connected
Specify message for notice of connection completed
Error message
Specify message to notify error from Proxy server
