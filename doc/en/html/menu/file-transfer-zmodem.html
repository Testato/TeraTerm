ZMODEM ([File] Transfer menu)

Transfers files between the PC and the host with the ZMODEM protocol.

Receive
Receives files from the host. Activate the send command of the host's ZMODEM before executing this command.
Send...
Sends files to the host.
Tera Term would call the host's ZMODEM "rz" command before sending ZRQINIT message.
