Quick-VAN ([File] Transfer menu)

Transfers files between the PC and the host with the Quick-VAN protocol.

Note: Quick-VAN protocol is used in the Japanese BBS "PC-VAN".

Receive
Receives files from the host. Activate the download command of the host before executing this command.
Send...
Sends files to the host. Activate the upload command of the host before executing this command.
