XMODEM Receive ([File] Transfer menu)

Receives a file from the host with the XMODEM protocol. Activate the send command of the host's XMODEM before executing this command.

"XMODEM Receive" dialog box

File Name

Enter the file name to be received.

Option

Checksum/CRC/1K
The XMODEM option. Select the same option as the host's XMODEM. If you don't know the host's option, try the Checksum option.
Binary
The binary file transfer option. If you want to receive a text file, don't select this option.
