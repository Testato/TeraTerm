Kermit ([File] Transfer menu)

Transfers files between the PC and the host with the Kermit protocol.

Receive
Receives files from the host. Activate the Send command of the host's Kermit before executing this command.
Get...
Receives files from the host, which is in the Kermit server state.
Send...
Sends files to the host. Activate the Receive or Server command of the host's Kermit before executing this command.
Finish
Exits the Kermit server state of the host.
