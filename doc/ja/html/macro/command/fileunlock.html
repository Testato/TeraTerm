fileunlock

開いているファイルのロックを解除する。 

fileunlock <file handle>

解説

すでに開いているファイルハンドル <file handle> を指定すると、ファイル全体のロックを解除する。

ロックの解除に成功するとシステム変数 result が 0 に、失敗すると 1 となる。

例

fileopen fhandle 'test.txt' 0
filelock fhandle
if result!=0 then
	messagebox 'error' 'hoge'
endif
filewrite fhandle 'sample'
pause 60
fileunlock fhandle
if result!=0 then
	messagebox 'error' 'hoge'
endif
fileclose fhandle
