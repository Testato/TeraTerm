expandenv

環境変数を展開する。

形式

expandenv <strvar> [<strval>]

パラメータ

文字列変数 <strvar>
第2引数を渡さない場合は、この引数で指定した文字列が展開される。
第2引数を渡した場合は、この引数は無視される。
文字列 <strval> (オプション)
この引数を指定した場合、この引数で指定した文字列が展開される。

返り値

文字列変数 <strvar>
展開後の文字列が格納される。

解説

渡された文字列に含まれる環境変数を展開する。

例

str = "%WINDIR%\notepad.exe"
expandenv str
messagebox str "env"

expandenv str "%WINDIR%\notepad.exe"
messagebox str "env"
