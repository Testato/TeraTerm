Quick-VAN のヒント

Quick-VAN は、PC-VAN という BBS で用いられているファイル転送プロトコルです。Quick-VAN は、7 bit 回線では使用できません。

シリアルポートの場合

Data が 7 bit だと、 Quick-VAN が使えません。[Setup] Serial port で Data を 8 bit にしてください。モデム(ホスト)側の設定も同じにしてください。

Telnet の場合

PC-VAN に接続する場合は、必ずバイナリオプションを指定してください (1995年12月現在)。バイナリオプションで接続するには、

  • コマンドラインに /B オプションをつけるか、
  • Tera Term 設定ファイルを書き換えてください。
    → Telnet のバイナリオプション

また、次の設定をすると転送速度が改善する場合があります。
→ Quick-VAN の window size

どうしても正常に転送できない場合は、ホスト側の Quick-VAN コマンドを正しく指定していることを確認した上で、 Quick-VAN のログを取り、作者に送ってください。
