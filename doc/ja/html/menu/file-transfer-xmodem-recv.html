XMODEM Receive ([File] Transfer メニュー)

XMODEM プロトコルでファイルを受信します。 その前にホスト側の XMODEM の送信コマンドを起動してください。

"XMODEM Receive" ダイアログボックス

File Name

受信するファイルのファイル名を入力または選択してください。

Option

Checksum/CRC
エラー検出の種類 (Checksum/CRC) を選択してください。
パケットサイズ(1K) は送信側でのみ指定できます。
Binary
バイナリファイルを受信する場合は選択してください。 逆にテキストファイルを受信する場合は選択しないでください。
