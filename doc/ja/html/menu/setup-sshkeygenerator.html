SSH Keygenerator ([Setup] メニュー)

"TTSSH: Key Generator" ダイアログボックス

詳しい解説はSSH 接続の「キーの生成」 をご覧下さい。

Key type
生成する鍵の種類を指定します。
RSA1
SSH1用 RSA鍵を生成します。
RSA
SSH2用 RSA鍵を生成します。
DSA
SSH2用 DSA鍵を生成します。
Key Bits
生成する鍵の鍵長を指定します。
Generate
鍵を生成します。
Key passphrase
生成した秘密鍵に付けるパスフレーズを指定します。
Confirm passphrase
秘密鍵に付けるパスフレーズの確認をします。 Key passphraseと同じものを入力してください。
Save public key
生成した公開鍵を保存します。
Save private key
生成した秘密鍵を保存します。
